Ausführliche Beschreibung

Mit Hilfe der Klasse QxSqlScript kann eine Datei mit SQL-Anweisungen ausgeführt werden.

Die Klasse QxSqlScript stellt Methoden zum Ausführen einer Datei mit SQL-Anweisungen bereit.

Dokumentation der Aufzählungstypen

enum cutex::QxSqlScript::Error
Aufzählungswerte
NoError

FileOpenError 

Die Datei konnte nicht geöffnet werden.

ConnectionError 

Es besteht keine Verbindung zur Datenbank.

StatementError 

Beim Ausführen einer SQL-Anweisung ist ein Fehler aufgetreten.
